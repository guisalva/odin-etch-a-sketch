# Odin - Etch a sketch

![Status Badge](https://img.shields.io/badge/status-concluded-brightgreen?style=for-the-badge)

This project was built to practice core concepts from the **Foundations Course** of [The Odin Project](https://www.theodinproject.com).

It is a simple sketchpad that you can easily change the grid size of the sketch.

## 🛠️ Tech Stack

- HTML
- CSS
- JS

## 🚀 Installation and Usage

Clone the repository:

```bash
git clone https://github.com/guisalva/odin-etch-a-sketch.git

cd odin-etch-a-sketch
```

Run the project:

1. Open `index.html` in your preferred browser;
2. Or use an extension like Live Server for a better development experience;
